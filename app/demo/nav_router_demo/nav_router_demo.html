导航

关键词:Route(路由), Navigation(导航),

步骤

  1. 在main.js中添加import { ROUTER_PROVIDERS } from '@angular/router';和bootstrap(AppComponent, [ ROUTER_PROVIDERS, ]);
  2. 在index.html中的<head>中添加<base href="/">标签
  3. 在添加导航的ts文件中添加import { Route, Routes, ROUTER_DIRECTIVES } from '@angular/router';和directives: [ ROUTER_DIRECTIVES, ],和@Routes([ new Route({ path: '/nav_router_demo', component: NavRouterDemo }), new Route({ path: '/demo2', component: Demo2Demo }), new Route({ path: '/demo3', component: Demo3Demo }), ]),其中还需要导入用到的组件import { NavRouterDemo } from './demo/nav_router_demo/nav_router_demo'; import { Demo2Demo } from './demo/demo2/demo2.demo'; import { Demo3Demo } from './demo/demo3/demo3.demo';
  4. 在ts对应的html文件中添加 <a [routerLink]="['nav_router_demo']"> 导航 </a> <a [routerLink]="['demo2']"> Demo2 </a> <a [routerLink]="['demo3']"> Demo3 </a> <main> <router-outlet></router-outlet> </main>
